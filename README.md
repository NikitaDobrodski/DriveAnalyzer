# DriveAnalyzer

Консольная утилита для диагностики дисков и USB-накопителей на Windows и Linux.

## Возможности

- Список всех подключённых дисков с информацией об использовании
- Подробная информация об устройстве: модель, серийный номер, интерфейс
- Тест скорости: последовательная запись/чтение и случайное чтение
- Диагностика здоровья диска через SMART

## Установка

Требуется Python 3.10+ и [Poetry](https://python-poetry.org/).

```bash
git clone https://github.com/NikitaDobrodski/DriveAnalyzer.git
cd DriveAnalyzer
poetry install
```

## Использование

### Список дисков

```bash
poetry run driveanalyzer list
```

Выводит таблицу всех подключённых дисков с размером, использованием, моделью и интерфейсом.

### Информация об устройстве

```bash
poetry run driveanalyzer info C:\
```

Показывает подробные данные о диске, на котором находится указанный путь.

### Тест скорости

```bash
poetry run driveanalyzer speed C:\ --size 512
```

Запускает три теста:
- Последовательная запись (по умолчанию 512 МБ)
- Последовательное чтение
- Случайное чтение (512 операций по 4 КБ)

Параметр `--size` задаёт размер тестового файла в МБ.

### Диагностика SMART

```bash
poetry run driveanalyzer health C:\
```

Считывает SMART-данные диска: статус, температуру, количество часов работы, переназначенные секторы и ошибки.

> На Windows требуются права администратора для чтения SMART-данных.

## Зависимости

| Пакет | Назначение |
|-------|-----------|
| psutil | Информация о дисках и использовании |
| rich | Красивый вывод в терминале |
| wmi | SMART-данные на Windows |
